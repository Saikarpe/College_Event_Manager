About CampusConnect

CampusConnect is a centralized web platform built as a mini-project for the Web Technologies course. It manages and displays events organized by student clubs across all departments.

🎯 Browse events from all departments in one place
🏛️ Discover student clubs and their activities
📅 Filter by department, status, or keyword search
🎫 Register for events via RSVP
🔐 Coordinator login to manage clubs & events (PHP + MySQL)
🗺️ GIS integration via OpenStreetMap & Leaflet.js

Tech Stack:

HTML5 CSS3 JavaScript (ES6+) PHP 8 Leaflet.js OpenStreetMap MySQL
